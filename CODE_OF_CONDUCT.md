# Code of Conduct

## Our commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Focus on ideas, not people.
- Assume positive intent and ask clarifying questions.
- Accept feedback with professionalism.

## Unacceptable behavior

- Harassment, threats, or discriminatory language.
- Personal attacks, trolling, or sustained disruption.
- Publishing private information without permission.

## Scope

This Code of Conduct applies to project spaces, issues, pull requests, and community discussions.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions not aligned with this Code of Conduct.

## Reporting

Report concerns by opening a private issue or contacting the maintainer through repository channels.
