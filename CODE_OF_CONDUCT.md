# Code of Conduct

## Our Pledge

We, as contributors and maintainers, commit to making participation in this project a respectful, harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, experience level, education, socioeconomic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Demonstrating empathy and kindness toward other people.
- Giving and receiving constructive feedback professionally.
- Focusing on technical outcomes rather than personal criticism.
- Taking responsibility for mistakes and correcting them quickly.
- Respecting differing viewpoints and lived experiences.

Examples of unacceptable behavior include:

- Harassment, intimidation, threats, or discrimination of any kind.
- Trolling, insulting/derogatory comments, and personal attacks.
- Sexualized language or imagery and unwelcome sexual attention.
- Publishing others' private information without explicit permission.
- Sustained disruption of issue threads, pull requests, or discussions.

## Scope

This Code of Conduct applies in all project spaces, including:

- GitHub issues, pull requests, and discussions
- Project documentation and review comments
- Any public or private communication representing the project

## Enforcement Responsibilities

Maintainers are responsible for clarifying and enforcing this Code of Conduct. They may remove, edit, or reject comments, commits, code, wiki edits, issues, and other contributions that violate this policy.

## Enforcement Guidelines

When a violation is reported, maintainers will review and respond in a fair, timely, and proportionate manner. Possible actions include:

1. Correction: private or public warning with a request to change behavior.
2. Warning: formal warning for continued or severe behavior.
3. Temporary ban: temporary loss of participation privileges.
4. Permanent ban: permanent removal from participation.

## Reporting

If you experience or witness unacceptable behavior:

- Use GitHub Security Advisories for sensitive reports:
  `https://github.com/dhanushk-offl/hallx/security/advisories/new`
- For non-sensitive concerns, contact maintainers via repository channels.

All reports will be handled as confidentially as possible.

## Attribution

This policy is informed by common open-source community standards, including the Contributor Covenant.
